# Paint-Project
Paint Brush Project
# Abstract 
Paint Brush is a Jframe based application that enables the user to draw basic shapes with 
different colors and line strokes. In addition, the user should have the ability to clear all the 
drawings or erase some parts of them. The user would be happy if the applications 
supported Undo operations. 

# Features:

1. **Colors Buttons** : The user should press one of these buttons before drawing to draw 
using this color.  

  **• Red**
  **• Green**
  **• Blue**

2. **Shapes Buttons**: The user should press one of these buttons before the drawing to 
draw a shape by dragging from a start point to an end point. The shape should be 
drawn while dragging till the user release the mouse button.

  • Rectangle
  • Oval 
  • Line

4. **Free Hand Button**: The user should press on this button to doddle while dragging.
   
5. **Eraser Button**: The user should press on this button be able to erase parts of the 
drawings.

6. **Clear All Button**: The user should press on this button to clear all of the drawings 
from the drawing area.

7. **Dotted Checkbox**: If checked it lets the shapes line stroke to be dotted. If not 
checked, it lets the shapes line strokes to be solid.

8. **Filled Checkbox**: If checked, it lets the Rectangle or Oval shapes that will be drawn 
after to be filled with the pre-selected color. If not check, it lest the shapes to be 
drawn as outlines only.

9. **Undo Button**: It lets the user undo the previous action(s)

10. **Save Button**: It enables the user to save his drawings to a file as image.
    
11. **Open Button**: It enables the user to load an image to the drawing area.
